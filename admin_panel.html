Connect to Vercel

Before you can use the admin panel, deploy the fpi-visitor-log project to Vercel and enter your app URL below. This only needs to be done once.

1
Deploy to Vercel Push the fpi-visitor-log repo to GitHub, then import it on vercel.com. Add a Postgres database from the Vercel dashboard.
2
Run the schema In your Vercel Postgres console, run the contents of schema.sql to create the tables.
3
Paste your Vercel app URL below Copy the URL Vercel assigned your project (e.g. https://fpi-visitor-log.vercel.app) and paste it here.
